## Build instructions


* Configurable theme mode
* Last modified date for posts
* Table of Contents
* Automatically recommend related posts
* Syntax highlighting
* Search
* Disqus Comments
* Google Analytics
* GA Pageviews reporting (Advanced)
* SEO and Performance Optimization
